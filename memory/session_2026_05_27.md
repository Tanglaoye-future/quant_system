## 关键技术洞察

### 1. macOS TCC（Full Disk Access）sudo 也绕不过

launchd 自 2026-05-15 报 `Operation not permitted`，是 TCC 拦截 bash 读 `~/Documents/` 保护目录。**sudo 无解**：
- TCC.db 受 SIP 保护
- `tccutil` 没有 grant 命令
- 唯一办法：GUI 加权限 / 移仓库出 Documents / 接受手动跑

用户选择 **C. 每天手动跑**（命令 `cd ~/Documents/projects/quant_system && ./deploy/run_daily.sh --no-options`）。

### 2. "换 universe 救美股策略"已证伪

| | NASDAQ100 mom-only | SP500 mom+quality |
|---|---|---|
| Sharpe | -0.22 | -0.18 |
| 胜率 | 42.5% | **37.5%** ↓ |
| trades | 273 | 365 |
| Admission | FAIL | FAIL |

**胜率反而降**是关键信号：等权 momentum + ATR trail stop 在美股 2022-2026 大盘成长占主导环境**结构性失效**，跟 universe 大小无关。后续若再有人想做美股 momentum，**先指向 [[sp500_negative_2026-05]] 跳过这两条已证伪路径**（换 universe / 加因子微调），评估是否做：
- B. 大改策略（市值加权 / 年度持有 / regime filter；1-2 周）
- C. 缩窄到 NASDAQ100 top-20 by market cap（0.5 day）
- A. 接受现状（QQQ 15% 被动 + options 期权，已是 [[deployment_plan_2026-05]] 当前方案）

### 3. fundamentals 接入是必要不充分条件

US 4y 在 yaml 权重全 0 时跑跟 momentum-only 一致；接入数据虽然 cache 全落地，但策略 yaml 不用它就等于没接。**任何后续美股因子实验前，先检查策略 yaml `factors.weights` 是否真的 enable 了 fundamentals 因子**。
