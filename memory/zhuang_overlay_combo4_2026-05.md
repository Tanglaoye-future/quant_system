## 关键洞察

1. **zhuang L4-combo4 在 2020-2026 单资产 Sharpe 2.35** — 之前 6y Sharpe 1.63 是更长窗口含早期低效区。
   2020 后窗口正是庄股策略表现最好的区间（疫情后小盘行情）。

2. **波动极低 (2.77%)** 是杀手锏 — zhuang 现金大部分时间不在场，仅触发吃货期入场 + 短持仓 (avg 4.4 天)，
   所以全周期波动远小于持续暴露的动量策略。

3. **相关性近零** — 因子完全独立于动量、价值、宏观资产。这是非投机者的"圣杯组合资产"。

4. **6y verify Sharpe 1.63 与单资产 2.35 的差异** — 前者用 2020-2026 完整窗口的回测年化指标，
   后者用同窗口的每日 returns 计算。差异来自 backtester 内 metrics.json 用的标准化方法不同
   （Sharpe 计算分母可能含 cash drag）。两者方向一致，本分析重算更严谨。

**Why:** zhuang Sharpe 从 0.94 → 1.63 (6y) / 2.35 (2020-2026 单资产)，6-asset overlay 重算证实
组合 Sharpe 提升 1.91 → 2.21 (zhuang 25%)；DD 7.6% → 5.1%。
**How to apply:** 实盘部署时 zhuang 占比从原 10% 提至 **20-25%**；deployment_plan_2026-05.md 需更新。
