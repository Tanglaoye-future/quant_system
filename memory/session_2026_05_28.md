## 三个值得记住的发现

1. **zhuang 是组合核心**：单 Sharpe 1.35 / DD -3.4% / 与万物 ≈0 相关。grid 想给 40%（cap binding，解开想到 55%）。是熊市压舱石（2022 把组合从亏损翻成盈利）。
2. **捕获 A_mr stale-data**：grid 初用 5/15 pre-DuckDB A_mr 曲线偏乐观；用 fresh post-DuckDB 重验 v5 不变。教训：复用回测产物看生成日期 + DuckDB 迁移(5/18)是分水岭。
3. **Options BCS 是"更好的 QQQ"**：BS 近似 Sharpe 1.36 > QQQ 0.92；7-asset grid 直接换掉 QQQ（+0.16 Sharpe，上界）。升核心 slot 需真实 fill 验证。
